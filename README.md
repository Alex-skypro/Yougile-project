# Автоматизация тестов для YouGile

Этот проект содержит автоматизированные UI и API тесты для веб-приложения YouGile, созданные в рамках курсовой работы.

## Ссылка на ручное тестирование

(Опишите кратко папки `pages/`, `api/`, `tests/`, `config/`)

## Предварительные требования
1. Python 3.9+
2. Установленный pip
3. Файл с переменными окружения `.env` (скопируйте `.env.example`)
